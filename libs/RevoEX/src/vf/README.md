# PrFILE2 (VF mod)

This is a decompilation of the PrFILE2 library developed by eSOL, but modified to be used for the VF library.

Based on the VF decompilation from [TFP2](https://github.com/koopthekoopa/TFP2/tree/main/src/vf) and [Petari](https://github.com/SMGCommunity/Petari/blob/master/src/RVL_SDK/vf) (which uses the a very similar version)
